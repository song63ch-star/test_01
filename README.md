# 🐍 스네이크 게임 (Snake Game)

HTML **단일 파일**로 제작된 클래식 **스네이크 게임**입니다.  
CSS와 JavaScript가 모두 포함되어 있어 파일 하나만으로 바로 실행 가능합니다.

---

## 🎮 주요 기능

- **화살표 키(↑ ↓ ← →)** 로 이동
- **Space**: 일시정지 / 재개
- **R**: 게임 재시작
- **벽 충돌 / 자기 몸 충돌 시 게임 오버**
- **점수 & 최고 점수(LocalStorage 저장)**
- 부드러운 움직임을 위한 `requestAnimationFrame` 기반 루프
- 반응형 UI · 픽셀 스타일 게임 화면

---

## 📁 파일 구성

- `snake.html`  
  → HTML, CSS, JS가 통합된 단일 실행 파일

---

## 🚀 실행 방법

1. 프로젝트 파일 다운로드 또는 복사  
2. `snake.html` 파일을 브라우저에서 실행  
3. 바로 게임 플레이 가능!

---

## ⚙️ 조작법

| 키 | 기능 |
|----|------|
| ↑ ↓ ← → | 이동 |
| Space | 일시정지 / 재개 |
| R | 재시작 |

---

## 🧩 커스터마이즈

- **게임 속도 변경** → `TICK_MS` 값 조절  
- **보드 크기 변경** → `GRID` 값 변경  
- **벽 통과 모드(랩 모드)** 구현 가능
